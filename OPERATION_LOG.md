## 本地测试使用方法

### 1. 启动 Mock 服务器
```bash
cd miniprogram-local-test/server
npm install
npm start
```

### 2. 打开测试页面
浏览器访问：http://localhost:3001/test.html

### 3. 模拟两人配对测试
1. 打开浏览器窗口 A，点击"登录测试账号"
2. 打开浏览器窗口 B（无痕模式），点击"登录测试账号"
3. 在窗口 A 生成邀请码
4. 在窗口 B 输入邀请码完成配对
5. 双方开始写信测试

### 4. 查看数据状态
访问：http://localhost:3001/api/stats

### 5. 清空测试数据
```bash
curl -X POST http://localhost:3001/api/clear
```
